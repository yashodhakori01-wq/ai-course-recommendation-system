## ✨ Features

✅ **Interactive User Input** - Collects student name, age, education level, and interests

✅ **7 Career Paths** - Covers diverse domains from IT to healthcare

✅ **Personalized Recommendations** - Suggests courses based on selected interest

✅ **Course Details** - Provides duration and brief description for each course

✅ **User-Friendly Interface** - Clear menus and formatted output

---

## 🎯 How It Works

1. **User Input Phase**
   - Student enters their name and age
   - Selects their education level (5 options)
   - Chooses their career interest area (7 categories)

2. **Processing Phase**
   - System identifies the selected interest category
   - Retrieves relevant course recommendations

3. **Output Phase**
   - Displays personalized student profile
   - Shows recommended courses with duration and description

## 🚀 Installation & Setup

### Prerequisites
- C compiler (GCC recommended)
- Linux/Unix/Windows environment
- Basic terminal knowledge

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/yashodhakori01-wq/ai-course-recommendation-system.git
   cd ai-course-recommendation-system
   ```

2. **Compile the program**
   ```bash
   gcc course_recommendation.c -o course_recommender
   ```

3. **Run the program**
   ```bash
   ./course_recommender
   ```
